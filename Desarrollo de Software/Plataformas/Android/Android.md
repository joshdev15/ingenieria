#development #android #mobile

Android es el sistema operativo mas usado del mundo para dispositivos moviles.

## Características

| Característica | Descripcion |
|----------------|-------------|
| **Lenguaje principal** | Kotlin (tambien Java) |
| **UI** | XML o Jetpack Compose |
| **Arquitectura** | MVVM, Clean Architecture |
| **Build system** | Gradle |
| **IDE** | Android Studio |

## Componentes Principales

- **Actividades** - Pantallas individuales
- **Fragmentos** - Componentes de UI modulares
- **Servicios** - Tareas en background
- **BroadcastReceivers** - Respuesta a eventos del sistema
- **ContentProviders** - Compartir datos entre apps
