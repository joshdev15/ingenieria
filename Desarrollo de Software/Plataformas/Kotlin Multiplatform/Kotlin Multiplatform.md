#development #kotlin #multiplatform

Kotlin Multiplatform (KMP) permite compartir codigo entre diferentes plataformas manteniendo codigo nativo donde se necesita.

## Características

| Característica | Descripcion |
|----------------|-------------|
| **Compartido** | Codigo comun para multiples plataformas |
| **Nativo** | Compila a codigo nativo de cada plataforma |
| **Interoperable** | Usa APIs nativas de cada plataforma |
| **Productivo** | Reduce codigo duplicado |

## Plataformas Soportadas

- **JVM** - Android, Backend, Desktop
- **Native** - iOS, macOS, Windows, Linux
- **Web** - JavaScript, WebAssembly
- **WASM** - WebAssembly

## Casos de Uso

- Compartir logica de negocio
- Compartir modelos de datos
- Compartir validacion
- Compartir codigo de red/base de datos
