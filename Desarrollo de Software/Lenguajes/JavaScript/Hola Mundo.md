#development #javascript

Ejemplo básico de "Hola Mundo" en JavaScript.

## Browser Console

```javascript
console.log("Hola mundo!");
```

## Node.js

```javascript
// hola.js
console.log("Hola mundo!");
```

```bash
node hola.js
```

## En HTML

```html
<!DOCTYPE html>
<html>
<body>
    <script>
        console.log("Hola mundo!");
        document.write("Hola mundo!");
    </script>
</body>
</html>
```

## Análisis

| Ambiente | Método |
|----------|--------|
| Browser | `console.log()`, `document.write()` |
| Node.js | `console.log()` |
| HTML | `<script>` tag |
