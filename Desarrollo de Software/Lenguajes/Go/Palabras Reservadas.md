#development #go

Palabras reservadas en Go (no pueden usarse como identificadores):

## Funciones Basicas

| Keyword | Descripcion |
|---------|-------------|
| `break` | Sale de un bucle |
| `continue` | Continua siguiente iteracion |
| `fallthrough` | Continua al siguiente caso en switch |
| `goto` | Salta a una etiqueta |
| `return` | Devuelve valor desde funcion |

## Declaraciones

| Keyword | Descripcion |
|---------|-------------|
| `const` | Define una constante |
| `func` | Define una funcion |
| `interface` | Define una interfaz |
| `map` | Define un map |
| `package` | Define un paquete |
| `struct` | Define una estructura |
| `type` | Define un tipo |
| `var` | Define una variable |

## Control de Flujo

| Keyword | Descripcion |
|---------|-------------|
| `if` | Condicional |
| `else` | Alternativa |
| `for` | Bucle |
| `range` | Iterar sobre slice/map/string |
| `select` | Espera de canales |
| `switch` | Seleccion multiple |

## Funciones Incorporadas

| Funcion | Descripcion |
|---------|-------------|
| `append` | Agrega elemento a slice |
| `cap` | Obtiene capacidad de slice |
| `close` | Cierra un canal |
| `copy` | Copia datos entre slices |
| `delete` | Elimina de un map |
| `len` | Longitud de slice/string/map |
| `make` | Crea slice/map/canal |
| `new` | Crea puntero |
| `print/println` | Imprime en consola |
