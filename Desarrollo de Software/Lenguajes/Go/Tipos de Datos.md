#development #go

Go tiene tipos de datos estrictos y bien definidos.

## Tipos Numericos

### Enteros (Con signo)
| Tipo | Bits |
|------|------|
| `int` | 32/64 |
| `int8` | 8 |
| `int16` | 16 |
| `int32` | 32 |
| `int64` | 64 |

### Enteros (Sin signo)
| Tipo | Bits |
|------|------|
| `uint` | 32/64 |
| `uint8` | 8 |
| `uint16` | 16 |
| `uint32` | 32 |
| `uint64` | 64 |

### Punto Flotante
| Tipo | Bits |
|------|------|
| `float32` | 32 |
| `float64` | 64 |

## Otros Tipos

| Tipo | Descripcion |
|------|-------------|
| `bool` | true/false |
| `string` | Secuencia Unicode |
| `byte` | Alias de uint8 |
| `rune` | Punto codigo Unicode |
| `complex64` | Complejo 64 bits |
| `complex128` | Complejo 128 bits |
| `uintptr` | Puntero a direccion memoria |
| `any` | Alias de interface{} (cualquier tipo) |
