#development #go

Go es multiplataforma (Linux, Mac, Windows) con una instalacion sencilla.

## Instalacion

1. Descargar el paquete desde [go.dev/dl](https://go.dev/dl)
2. Ejecutar el instalador segun el sistema operativo

## Variables de Entorno

Es fundamental configurar:

| Variable | Descripcion |
|---------|-------------|
| **GOPATH** | Ruta del espacio de trabajo |
| **GOROOT** | Ruta donde esta instalado Go |

## Verificar Instalacion

```bash
go version
go env
```
