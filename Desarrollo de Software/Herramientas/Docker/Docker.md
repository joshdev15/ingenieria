#development #tool #docker

Docker es una plataforma para desarrollar, enviar y ejecutar aplicaciones en contenedores.

## Características

| Característica | Descripcion |
|----------------|-------------|
| **Contenedores** | Aislamiento ligero de procesos |
| **Imagenes** | Templates readonly para contenedores |
| **Dockerfile** | Receta para crear imagenes |
| **Docker Hub** | Registro publico de imagenes |
| **Compose** | Orquestacion multi-contenedor |

## Conceptos Clave

- **Contenedor**: Proceso aislado con su propio filesystem
- **Imagen**: Template inmutable para crear contenedores
- **Volume**: Almacenamiento persistente
- **Network**: Comunicacion entre contenedores
