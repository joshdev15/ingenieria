#development #tool #grpc

gRPC es un framework de Remote Procedure Call (RPC) de alto rendimiento desarrollado por Google.

## Características

| Característica | Descripcion |
|----------------|-------------|
| **Protocol Buffers** | Serializacion binaria eficiente |
| **HTTP/2** | Multiplexacion y streaming |
| **Generacion de codigo** | stubs automatica multi-lenguaje |
| **Bidireccional** | Comunicacion cliente-servidor streaming |
| **Fuertemene tipado** | Contratos definidos |

## Usos Comunes

- Microservicios
- APIs de alto rendimiento
- Comunicacion entre servicios
- Streaming de datos
- Mobile apps
