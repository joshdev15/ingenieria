#engineering #project-management #tools

Notion es una herramienta todo-en-uno que combina notas, bases de datos y gestión de proyectos.

## Estructura Flexible

Notion se basa en **pages** que pueden contener cualquier cosa:

```
Page (Proyecto)
├── Subpages (secciones)
├── Blocks (contenido)
│   ├── Text
│   ├── Heading
│   ├── List
│   ├── Table/Database
│   ├── Board (Kanban)
│   ├── Calendar
│   ├── Code
│   └── Image
└── Properties (metadatos)
```

## Bases de Datos (Databases)

Las bases de datos son como hojas de cálculo flexibles:

| Tipo de Vista | Descripción |
|---------------|-------------|
| **Table** | Vista de tabla tradicional |
| **Board** | Tablero Kanban |
| **Gallery** | Tarjetas visuales |
| **List** | Lista de filas |
| **Calendar** | Vista de calendario |
| **Timeline** | Línea de tiempo |

### Ejemplo: Base de Tareas

```
Propiedades:
├── Status (Select: To Do, In Progress, Done)
├── Priority (Select: High, Medium, Low)
├── Due Date (Date)
├── Tags (Multi-select)
├── Assignee (Person)
└── Related Project (Relation)
```
