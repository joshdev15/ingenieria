#engineering #project-management #tools

Trello es una herramienta de gestión visual basada en tableros Kanban, simple e intuitiva.

## Concepto Básico

Todo se organiza en **boards** (tableros) con **lists** (columnas) y **cards** (tarjetas).

```
Board: Proyecto Web
├── To Do
│   ├── Card: Diseñar mockups
│   └── Card: Configurar repo
├── In Progress
│   └── Card: Desarrollo frontend
└── Done
    └── Card: Reunión de kickoff
```

## Características

| Feature | Descripción |
|---------|-------------|
| **Boards** | Tableros para proyectos/áreas |
| **Lists** | Columnas (ej: To Do, Doing, Done) |
| **Cards** | Tareas con detalles |
| **Labels** | Etiquetas para categorizar |
| **Due dates** | Fechas de vencimiento |
| **Attachments** | Archivos adjuntos |
| **Power-Ups** | Plugins para funcionalidad extra |

## Power-Ups Populares

- **Calendar**: Vista de calendario
- **Automation**: Automatizar acciones
- **Slack**: Integración con Slack
- **Custom Fields**: Campos personalizados

## Pricing

- **Free**: Unlimited boards, 10 cards per board
- **Standard**: $5/usuario/mes (ilimitado)
- **Premium**: $10/usuario/mes (dashboard avanzado)
- **Enterprise**: $17.50/usuario/mes

## Cuándo Usar Trello

- Equipos pequeños/medianos
- Proyectos simples
- Preferir simplicidad sobre features
- Visibilidad del flujo de trabajo
