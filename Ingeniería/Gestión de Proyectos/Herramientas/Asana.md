#engineering #project-management #tools

Asana es una plataforma de gestión del trabajo que combina tareas, proyectos y colaboración de equipo.

## Vistas Principales

| Vista | Descripción |
|-------|-------------|
| **List** | Lista tradicional de tareas |
| **Board** | Tablero estilo Kanban |
| **Timeline** | Línea de tiempo visual (Gantt) |
| **Calendar** | Vista de calendario |
| **Dashboard** | Métricas y progresso |

## Estructura

```
Workspace
└── Team
    └── Project
        ├── Section/Group
        │   └── Tasks
        │       ├── Subtasks
        │       └── Attachments
        └── Tasks
```

## Características

| Feature | Descripción |
|---------|-------------|
| **Tasks** | Tareas con subtareas, fechas, descripciones |
| **Portfolios** | Vista de múltiples proyectos |
| **Goals** | Objetivos alineados con proyectos |
| **Forms** | Formularios para solicitudes |
| **Automations** | Reglas automáticas |
| **Integrations** | Slack, Google Drive, etc. |

## Forms

Los formularios permiten que otros envíen solicitudes de trabajo sin acceso al proyecto.

```
Formulario: Solicitud de Bug
├── Título
├── Descripción
├── Prioridad
├──截图/Pantallazo
└── Enviar → Se crea tarea automáticamente
```

## Pricing

- **Basic**: Free (para equipos)
- **Starter**: $10.99/usuario/mes
- **Advanced**: $24.99/usuario/mes
- **Enterprise**: Custom

## Diferencia con Trello

| Aspecto | Asana | Trello |
|---------|-------|--------|
| Complejidad | Mayor | Menor |
| Timeline | ✅ nativo | ⚠️ Power-Up |
| Goals | ✅ nativo | ❌ No |
| Formularios | ✅ nativo | ⚠️ Limitado |
