#engineering #project-management #tools

Linear es una herramienta de gestión de proyectos diseñada específicamente para equipos de desarrollo de software.

## Filosofía

- **Rápida**: Interfaz instantánea y fluida
- **Minimalista**: Solo lo esencial
- **Orientada a developers**: Keyboard-first, shortcuts

## Características

| Feature | Descripción |
|---------|-------------|
| **Cycles** | Iteraciones como sprints |
| **Projects** | Proyectos con múltiples ciclos |
| **Issues** | Tareas con tipos (bug, feature, task) |
| **Board** | Vista Kanban por estado |
| **List** | Vista de lista filtrable |
| **Timeline** | Planificación visual |
| **Automations** | Reglas automatizadas |

## Estructura

```
Workspace
├── Project
│   ├── Cycle (Sprint)
│   │   └── Issues
│   └── Issues
└── Views (Board, List, Timeline)
```

## Teclas de Atajo (Key Features)

```
Cmd + K: Quick switch
Cmd + Enter: Completar tarea
Cmd + Shift + P: Crear proyecto
Cmd + \: Buscar
```

## Integraciones

- GitHub, GitLab, Bitbucket
- Slack
- Figma
- VS Code (extensión)

## Pricing

- **Free**: Hasta 250 issues
- **Standard**: $8/usuario/mes
- **Plus**: $15/usuario/mes (cycles ilimitados)
- **Enterprise**: Custom
