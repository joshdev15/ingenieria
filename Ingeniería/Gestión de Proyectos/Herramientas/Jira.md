#engineering #project-management #tools

Jira es la herramienta de gestión de proyectos más popular para equipos ágiles y de desarrollo de software.

## Características Principales

### Tipos de Proyecto
- **Software**: Para equipos ágil/Scrum/Kanban
- **Business**: Para operaciones, marketing, HR
- **Work Management**: Para cualquier tipo de proyecto

### Funcionalidades Clave

| Feature | Descripción |
|---------|-------------|
| **Backlog** | Lista priorizada de historias/epics |
| **Sprints** | Iteraciones planificables |
| **Kanban Board** | Tablero visual de flujo |
| **Roadmap** | Planificación a largo plazo |
| **Reports** | Burndown, velocity, cumulative flow |
| **Automation** | Reglas automáticas para workflows |

## Estructura de Elementos

```
Project
├── Epic (gran feature)
│   ├── Story (historia de usuario)
│   ├── Story
│   └── Task (tarea específica)
└── Bug (defecto)
```

## Flujo de Trabajo (Workflow)

```
To Do → In Progress → In Review → Done
```

Puedes personalizar los estados y transiciones.

## Pricing

- **Free**: Hasta 10 usuarios
- **Standard**: $7.50/usuario/mes
- **Premium**: $14.50/usuario/mes
- **Enterprise**: Custom

## Uso Típico

1. Crear proyecto
2. Configurar backlog con historias
3. Planificar sprint
4. Actualizar tareas diariamente
5. Revisar y cerrar
