#engineering #tools #diagramas

Draw.io (diagrams.net) es una herramienta gratuita de diagramación visual.

## Características

| Feature | Descripción |
|---------|-------------|
| **Interfaz Visual** | Arrastrar y soltar elementos |
| **Templates** | Plantillas predefinidas |
| **Formas** | UML, flowcharts, redes, etc. |
| **Exportar** | PNG, SVG, PDF, URL |
| **Gratuito** | Sin costo |
| **Offline** | Funciona sin internet |

## Tipos de Diagramas Soportados

- Diagramas de flujo
- Diagramas UML (todos los tipos)
- Mapas mentales
- Diagramas de red
- Diagramas de arquitectura
- Wireframes

## Uso

1. Ir a [draw.io](https://draw.io)
2. Seleccionar dónde guardar (Google Drive, OneDrive, device)
3. Crear nuevo diagrama
4. Arrastrar elementos desde la barra lateral
5. Conectar con flechas
6. Exportar o guardar

## Ventajas

- Gratis y sin registro necesario
- Intuitivo para principiantes
- много templates disponibles
- Exporta a múltiples formatos

## Desventajas

- Menos automatización que código
- Puede ser lento para diagramas muy grandes
- No es colaborativo en tiempo real (versión gratuita)
