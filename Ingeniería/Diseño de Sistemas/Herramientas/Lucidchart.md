#engineering #tools #diagramas

Lucidchart es una herramienta profesional de diagramación con colaboración en tiempo real.

## Características

| Feature | Descripción |
|---------|-------------|
| **Editor Visual** | Arrastrar y soltar |
| **UML** | Diagramas UML especializados |
| **Colaboración** | Tiempo real, comentarios |
| **Integrations** | Herramientas de desarrollo |
| **Templates** | Centenas disponibles |
| **Export** | PDF, PNG, SVG, Visio |

## Diagramas UML Soportados

- Diagrama de clases
- Diagrama de secuencia
- Diagrama de casos de uso
- Diagrama de estados
- Diagrama de componentes
- Diagrama de actividades

## Pricing

- **Free**: Hasta 3 documentos
- **Pro**: $7.95/usuario/month
- **Team**: $12.50/usuario/mes (colaboración)
- **Enterprise**: Custom

## Integrations

- Google Docs, Slides, Sheets
- Confluence, Jira
- Slack
- Microsoft Teams
- VS Code (extensión)

## Diferencia con Draw.io

| Aspecto | Lucidchart | Draw.io |
|---------|-------------|---------|
| UML | Especializado | Básico |
| Colaboración | ✅ tiempo real | Limitado |
| Pricing | Freemium | Gratis |
| Integrations | Más | Menos |
